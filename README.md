# ESP8266 Weather Visualiser

## Pins

### ESP8266 to OLED Display

3V - VCC

GND - GND

D1 - SCK

D2 - SDA

### ESP8266 to NeoPixel Ring

VU - 5V

GND - GND

D6 - DI(Data Input)

DO(Data Output) does not need to be connected
